ANEURISK DATA ANALYSIS

We analyze the aneurisk data set containing decriptive variables of patients showing ruptured and unruptured aneurisms.

1 Intro

In the next section we make an exploratory descriptive statistical analysis, to probe the main characteristics in the data. In the following section we try out some preliminary models to explore correlations that may prove useful for future, more realistic models.
